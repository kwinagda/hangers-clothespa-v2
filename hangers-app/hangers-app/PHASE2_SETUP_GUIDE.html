Part A — Set Up WhatsApp Meta API
One-time setup. Takes ~30 minutes.
REQUIRED FIRST
💬  Why WhatsApp? Your customers already have WhatsApp. OTPs arrive instantly, are more trustworthy than SMS, and you don't need a separate SMS provider. The Meta Cloud API is free for the authentication template category.
1
Create a Meta Business Account

Go to business.facebook.com → Click "Create Account" → Fill in your business name "Hangers Clothes Spa" and your details. You'll need to verify your identity.

2
Create a Meta Developer App

Go to developers.facebook.com → My Apps → Create App → Choose "Business" type → Name it "Hangers" → Link to your Business Account from Step 1.

3
Add WhatsApp Product to Your App

In your app dashboard → "Add Product" → Find "WhatsApp" → Click "Set Up". This gives you access to the WhatsApp Cloud API.

4
Get Your Phone Number ID

In your app → WhatsApp → API Setup → You'll see a test phone number. For production, you need to add your actual business WhatsApp number (+91 7977417014). Click "Add phone number" and verify it.

WHERE TO FIND IT
App Dashboard → WhatsApp → API Setup → "Phone Number ID" (15-digit number like: 123456789012345) → Copy this — it goes in META_WA_PHONE_NUMBER_ID
5
Generate a Permanent Access Token

Temporary tokens expire after 24 hours. For production you need a permanent one via a System User:

STEPS TO GET PERMANENT TOKEN
business.facebook.com → Settings → Users → System Users → Create System User (name: "hangers-api", role: Admin) → Generate New Token → Select your App → Check permissions: whatsapp_business_messaging whatsapp_business_management → Generate Token → Copy and save it → This goes in META_WA_ACCESS_TOKEN
6
Create the OTP Message Template

In business.facebook.com → WhatsApp Manager → Account Tools → Message Templates → Create Template:

TEMPLATE SETTINGS — FILL IN EXACTLY AS SHOWN
Template Name: hangers_otp Category: AUTHENTICATION Language: English Header: (none — leave blank) Body: {{1}} is your OTP for Hangers Clothes Spa. Valid for 10 minutes. Do not share. Footer: (optional) "Care in Every Clean" Buttons: (optional) One-tap "Copy Code" button
⏱  Template approval usually takes 5–30 minutes for AUTHENTICATION category. You'll get an email when it's approved. Do not proceed until approved.
7
Update Your .env File
.env — ADD THESE 4 LINES
META_WA_PHONE_NUMBER_ID="your_15_digit_phone_number_id" META_WA_ACCESS_TOKEN="your_permanent_system_user_token" META_WA_OTP_TEMPLATE="hangers_otp" META_WA_BUSINESS_ACCOUNT_ID="your_waba_id"
8
Update Backend — Replace the old msg91.service.js

Copy the new whatsapp-otp.service.js into your backend's src/services/ folder, and replace src/controllers/auth.controller.js with the updated version. Also update your .env.example.

ℹ️  No MSG91 needed anymore. If MSG91 keys are still in your .env, just leave them — the new service completely ignores them.

Part B — Set Up the Customer App
Install Expo and run on Android
EXPO / REACT NATIVE
⬇️

Install Expo CLI

Open your terminal and run:

TERMINAL
npm install -g expo-cli eas-cli npm install -g expo
📁

Navigate to the app folder and install packages

TERMINAL
cd Desktop/hangers-app # or wherever you placed it npm install # installs all React Native packages (~2 min)
🔌

Set the backend URL in api.ts

Open src/services/api.ts. Find the BASE_URL line and update it to your computer's local IP address (for testing on a real Android phone):

src/services/api.ts — LINE 14
// Find your IP: run `ipconfig` (Windows) or `ifconfig` (Mac) in terminal // Android Emulator → use 10.0.2.2 // Real Android phone → use your computer's WiFi IP (e.g. 192.168.1.10) const BASE_URL = 'http://192.168.1.10:5000/api/v1'; // Replace 192.168.1.10 with your actual IP — both devices must be on same WiFi
📲

Install Expo Go on your Android phone

Open the Google Play Store on your Android phone → Search "Expo Go" → Install it. This lets you run the app instantly without building an APK.

🚀

Start the app

Run this in the hangers-app folder:

TERMINAL
npm start # or: npx expo start # A QR code will appear in the terminal. # Open Expo Go on your phone → Scan QR code # The app will load on your phone!
✅  You should see the animated Hangers splash screen, then the phone entry screen. Enter your number to get an OTP on WhatsApp!
🖼️

Add your Hangers logo (optional but recommended)

Place your logo files in src/assets/:

FILES NEEDED
src/assets/ ├── icon.png → 1024×1024 (app icon) ├── splash.png → 1284×2778 (splash background) ├── adaptive-icon.png → 1024×1024 (Android adaptive icon) └── hangers_logo.png → your existing logo file

To use the actual logo in the splash screen, open SplashScreen.tsx and replace the emoji View with:
<Image source={require('../assets/hangers_logo.png')} style={{'{'}}{'}'}width:110, height:110{'}'}} />
